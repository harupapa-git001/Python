外部ライブラリの使用は基本的に初回はインストールが前提ですが、既にデフォルトとして入っているライブラリもある可能性がありますので、下記のターミナルコマンドを実行して確認してください。

$ pip show モジュール名

#ex paperclipをimportしたい場合

$ pip show pyperclip

上記のコマンドでNo fileなどの回答になった場合は下記ターミナルコマンドでインストールします。

$ pip install モジュール名

または

$ pip3 install モジュール名（インターネット表記ではこちらが多いですが、環境によってpip3指定をすると正常にインストール出来ない事があるかもしれません。その場合は上記のインストールでお試しください）

----------------------------------------------------------------------------------

OPENPYXL.pyを実行する際はnya.xlsxを同じディレクトリに置くか、削除して実行してください。
